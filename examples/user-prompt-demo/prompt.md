# 用户提示词示例

请使用 `$draw-cs-research-figures` 为一篇人工智能论文绘制方法框架图。

论文方法名称：Confidence-Aware Multi-Agent Multimodal RAG。

研究内容：系统接收文本问题、图像/视频和领域文档。首先由多模态编码器生成视觉与文本 token，混合检索器同时访问向量数据库和知识图谱。规划智能体根据问题复杂度生成任务图，动态路由器把子任务分配给视觉分析智能体、知识图谱推理智能体和工具执行智能体。三个智能体通过共享记忆交换证据；验证智能体检查证据一致性、引用有效性和回答置信度。当证据不足时，验证智能体向规划智能体发出反馈并触发再次检索。最后，证据融合模块输出带引用的回答和置信度。

需要重点表现的创新：

1. 问题感知的动态智能体路由；
2. 多智能体共享记忆与证据交换；
3. 验证—重新规划反馈环；
4. 向量检索与知识图谱检索联合使用；
5. 输出回答、引用和不确定性。

绘图要求：采用从左到右的四分区布局；加入 token 条带、张量层叠、知识图节点、数据库、智能体专属图标、局部创新放大框和反馈虚线；标题较长时分成两行；所有普通文字保持水平；采用蓝、橙、紫、绿和红色的语义配色；整体风格适合 CVPR/ACL 论文，信息密度较高但不能出现文字裁切或连线穿过模块。输出 PNG，分辨率不低于 1800×1000，嵌入 300 DPI。
